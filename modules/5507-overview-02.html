This file was created by Steve Simon using R Markdown on 2020-09-24 and last updated on 2021-01-16. You can find the source code on my github repository, classes-medb.

Learning objectives

  1. To read delimited text files

  2. To read fixed width files

  3. To read files with variable names in the first row

  • filename
  • proc import
